# Portfolio Website

This is a static portfolio website for Alphawaves.

## Structure

- **/index/**: Home page
- **/about/**: About page (also used for Ideas/Now)
- **/portfolio/**: Portfolio/projects page
- **/assets/css/**: CSS files
- **/assets/img/**: Images and assets
- **/assets/js/**: JavaScript (optional)
- **/shared/**: Shared HTML components (header, footer)

## How to Use

Upload the contents of this directory to your web server. All pages are static HTML and CSS.
